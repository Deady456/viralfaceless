Setup GitHub

        
Fork repo dan set secret di GitHub.

        

          

            
➕ Langkah 1: Fork Repo Template

            
    
              
  1. Buka github.com/Deady456/sainstek
    2. 
              
  2. Klik tombol Fork di kanan atas
    3. 
              
  3. Beri nama repo: channel-kamu
    4. 
              
  4. Klik Create Fork
    5. 
            

          

          

            
🔒 Langkah 2: Set GitHub Secrets

            
Di repo kamu: Settings → Secrets and variables → Actions → New repository secret

            

              

                
SecretIsi

                
                  
GROQ_API_KEYDari GroqCloud (gratis)

                  
PEXELS_API_KEYDari Pexels (gratis)

                  
ELEVENLABS_API_KEYS(opsional) 20 key dipisah koma

                  
CLIENT_SECRET_B64Base64 client_secret.json

                  
TOKEN_CHANNEL_B64Base64 token YouTube

                
              

            

          

          

            
⚡ Langkah 3: Trigger Manual

            
Di repo kamu: Actions → Daily Video Upload → Run workflow. Cek apakah video terupload.

          

        

        

          
          
        

      

    


    
    

      

        
Authorize YouTube

        
Hubungkan repo ke channel YouTube kamu.

        

          

            
💻 Cara Authorize (PC)

            
git clone https://github.com/username/channel-kamu
cd channel-kamu
python -m pip install -r requirements.txt
python -m src.authorize
# Buka URL yang muncul, login Google, pilih channel

          

          

            
📱 Alternatif (HP + Termux)

            
pkg install python git
pip install google-auth-oauthlib google-api-python-client
git clone https://github.com/username/channel-kamu
cd channel-kamu
python -m src.authorize

          

          

            
Setelah authorize, hasilnya file token_channel.json. Encode ke base64 dan set jadi secret TOKEN_CHANNEL_B64 di GitHub.

          

          

            📄 Cara encode token ke base64
            
powershell -Command "[Convert]::ToBase64String([IO.File]::ReadAllBytes('token_channel.json'))"
